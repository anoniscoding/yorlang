Yorlang Programming Language

Yorlang is an esoteric programming language designed to write small programs using yoruba language constructs. It is the first of its kind and it can carry out the basic programming operations, alas! in yoruba language (a language spoken in some parts of Nigeria ).

Getting Started Contribute

Variable Declaration

Variables are created with keyword 'tí'. Since yorlang is a dynamic language, any variable created with this keyword can hold any type of value

Conditionals & Loops

There are various types of loops and conditionals in yorlang. They include fún (for loop), nígbàtí (while loop), yí (switch loop), sé (if block), and tàbí (else block).

Functions & Arrays

Functions are declared with keyword 'iṣé'. There's support for nested functions in yorlang. Arrays and multi-dimensional arrays are also included in yorlang.
